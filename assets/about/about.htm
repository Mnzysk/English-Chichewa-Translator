1.0       System Menu

 

English To Chichewa translator is tabbed application, which consists of 3 tabs (Figure 1). The first tab is the translate tab which is the home screen for the application where the actual translation is done. The second tab is the history tab which contains all the previous searches the user has performed. The third tab is the suggest tab where the user is able to suggest a translation which is then sent to the administrator for verification.All the tabs are bordered and are namedto allow distinguishing them easily, the active tab is underlined with a thick blue line.

 

Figure 1. Tabs of English To Chichewa translator

 

1.1  Translate tab

The Translate tab (Figure 2) consists of an input field, a button and a view text field. The input field allows the user to input the English phrase the desire to translate, it has a hint “Lembaniapa” for easy identification. In addition the “X” to the right of the text field is used to clear the input field.The “Translate” button initiates the translation process. Lastly, the view text field displays the translated text and has the hint “Tanthauzo la Chichewa” as the default output.

 

Figure 2.The Translate tab

 

1.2 History tab

The History tab (Figure 3) consists of a list view output field, and two clickable icons. The list view output is where the actual translation history is displayed. The list is expandable to allow the user to see the translation to that phrase. The thrash can icon (bottom left figure 3) allows the user to clear all history or just the selected phrases (figure 4). The refresh icon (bottom right figure 3) allows the user to refresh the history list.

 

Figure 3.The History tab

 

Figure 4.Delete options

 

1.3  Suggest Tab

TheSuggest  Gallery  tab  (Figure  5)  consist of three input text fields and a button. The first text field is the “English text”field, the second is “Mau a Chichewa” text field , the third and final text field is the “Comment” input text field. The only button is the submit “Submit”button.
